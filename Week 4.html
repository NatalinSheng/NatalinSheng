Week 4

Lego Brick

This week we were tasked to complete multiple tasks such as moedlling a lego brick, nametag, as well as a mug. When I was trying out the lego brick, one key feature that stood out to me was the pattern feature. This feature allowed me to duplicate the lego studs in the way I wanted as well as helped to to save time as I did not have to do it one by one.



NameTag

When I did the nametag, I learned that there was the 'text' feature which was something I never really thought that there was but it's something that was really helpful. I first started off by sketching a rectangle and adding the 2 circles to make the name tag hole, and then extracted it. Then I followed by using the 'shell' feature to shell out the part where I was going to put the text on, and then extracted that as well.



Stacked Cubes

I first started out by sketching a square, extruding it and then duplicating it into the stacked shape. Then to allow for the shell feature on the software to create the carving of the model. I was able to choose the thickness of the 'unshelled' parts. In the past i would resort to skecthing on a plane and extruding it, whenever I wanted to have a hole, but with this feature, it makes a lot more easier.



Mug

To start, I skechted a circle and extruded it to the desired height. Then to create a hole, I used the 'shell' feature on the software. To include the handle, I was not able to pick a already existing plane, so I had tto create a new plane on the side of the cup to allow the handle to stick onto the body. Once I managed to find the ideal plane, I was able to create the handle shape using the 'arc' tool to create the ideal handle shape.
